# Language Operator Agents

Specialized AI workflows for repetitive tasks in Language Operator development.

## Available Agents

- [changelog-writer](changelog-writer.md) - Creates user-focused changelog entries from git diffs and commit analysis

## Usage

To use an agent:
```
Use the changelog-writer agent to create a changelog entry for my recent changes
```

Agents analyze your specific context (git diffs, project files) and follow step-by-step processes to deliver consistent, high-quality output.
